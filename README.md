# Secure Document Holder

Secure iOS document vault with biometric (Face ID) protection.

## Features
- Per-file Face ID lock/unlock
- Import from Files, Photos, Camera
- Rename before saving via expandable FAB
- Automatic categorization
- Trash with Face ID access
- Dark vault design
- Hybrid PDF/Document viewer

## Agile Development
Followed full Agile methodology:
- Plan → Design → Develop → Test → Review → Deploy

Used feature branches to organize development:
- `feature/biometric-security`
- `feature/fab-import`
- `feature/document-management`

All branches pushed and visible in repository.
